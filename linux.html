🐧 Инструкции по установке игр на Linux

Пошаговые инструкции и советы по запуску игр EgorNetwork на Linux.

Установка через Wine/Proton

Большинство игр для Windows можно запустить на Linux через Wine или Proton (Steam Play).

  1. Установите Wine: sudo apt install wine (для Debian/Ubuntu)
  2. Или включите Proton в Steam: Настройки → Steam Play → Включить Steam Play
  3. Запустите установщик игры через Wine
  4. Следуйте инструкциям установщика

Проблемы и решения

Частые проблемы и способы их решения.

  • Игра не запускается: Попробуйте установить зависимости через winetricks
  • Нет звука: Проверьте настройки звука в Wine
  • Низкая производительность: Используйте Proton Experimental
